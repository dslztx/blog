еҺҹеӯҗж“ҚдҪңзұ»

еҸ‘иЎЁдәҺ 2020-07-30 жӣҙж–°дәҺ 2021-03-13 дё“йўҳдәҺ иҜӯиЁҖ пјҢ Java пјҢ Javaе№¶еҸ‘ Disqusпјҡ

жң¬ж–Үд»Ӣз»Қjava.util.concurrent.atomicеҢ…дёӢзҡ„еҺҹеӯҗж“ҚдҪңзұ»пјҢдҪҝз”Ёзҡ„JDKжҳҜJDK 8гҖӮ

дёҖгҖҒеүҚеҜјзҹҘиҜҶ

дёәзҗҶи§Јжң¬ж–ҮпјҢйңҖиҰҒзҡ„еүҚеҜјзҹҘиҜҶеҲ—иЎЁеҰӮдёӢпјҡ

  • volatileеҸҳйҮҸзҡ„еҶ…еӯҳиҜӯд№үпјҢеҸӮи§ҒгҖҠsynchronized-volatile-finalе…ій”®иҜҚгҖӢ
  • Unsafeзұ»дёӯзҡ„дёҖдәӣж–№жі•пјҢеҸӮи§ҒгҖҠUnsafeзұ»гҖӢпјҢе…·дҪ“жңүпјҡ
    • 3дёӘCASж“ҚдҪңж–№жі•пјҲcompareAndSwapIntпјҢcompareAndSwapLongе’ҢcompareAndSwapObjectпјү
    • 9дёӘput*Volatileж–№жі•е’Ң9дёӘget*Volatileж–№жі•
    • putOrderedInt/putOrderedLong/putOrderedObjectж–№жі•
  • еҺҹеӯҗж“ҚдҪңдёҺй”ҒпјҢеҸӮи§ҒгҖҠеҺҹеӯҗж“ҚдҪңдёҺй”ҒгҖӢ
  • finalе…ій”®иҜҚеҶ…еӯҳеҸҜи§ҒиҜӯд№үпјҢеҸӮи§ҒгҖҠsynchronized-volatile-finalе…ій”®иҜҚгҖӢ

дәҢгҖҒjava.util.concurrent.atomicеҢ…дёӢзҡ„еҺҹеӯҗж“ҚдҪңзұ»

еҸҜеҲҶжҲҗ4зұ»пјҡ

  • еҹәжң¬зұ»еһӢжҲ–иҖ…еј•з”Ёзұ»еһӢеҺҹеӯҗдҝ®ж”№ж“ҚдҪңзұ»
  • ж•°з»„е…ғзҙ еҺҹеӯҗдҝ®ж”№ж“ҚдҪңзұ»
  • еҜ№иұЎеӯ—ж®өеҺҹеӯҗдҝ®ж”№ж“ҚдҪңзұ»
  • еҺҹеӯҗи®Ўж•°/зҙҜеҠ ж“ҚдҪңзұ»

2.1гҖҒеҹәжң¬зұ»еһӢжҲ–иҖ…еј•з”Ёзұ»еһӢеҺҹеӯҗдҝ®ж”№ж“ҚдҪңзұ»

дёҖе…ұжңү6дёӘзұ»пјҢеҸҜеҲҶдёәдёӨзұ»пјҡ

  • з¬¬дёҖзұ»пјҡAtomicIntegerпјҢAtomicLongпјҢAtomicReferenceпјҢAtomicBooleanгҖӮжҲ‘д»¬зҹҘйҒ“Javaж•°жҚ®зұ»еһӢеҸҜеҲҶдёәвҖңеҹәжң¬зұ»еһӢпјҲbooleanпјҢbyteпјҢcharпјҢshortпјҢintпјҢfloatпјҢlongпјҢdoubleпјүвҖқе’ҢвҖңеј•з”Ёзұ»еһӢвҖқпјҢеңЁUnsafeзұ»дёӢжңү3дёӘCASж“ҚдҪңж–№жі•пјҲcompareAndSwapIntпјҢcompareAndSwapLongе’ҢcompareAndSwapObjectпјүпјҢзӣҙжҺҘеҜ№еә”3дёӘеҺҹеӯҗдҝ®ж”№ж“ҚдҪңзұ»пјҢеҸҰеӨ–йҖҡиҝҮдёҖе®ҡи®ҫи®ЎпјҲд»ҘintеһӢзҡ„вҖң1|0вҖқеҲҶеҲ«д»ЈиЎЁbooleanеһӢзҡ„вҖңtrue|falseвҖқпјүиҺ·еҫ—еҜ№еә”дәҺвҖңbooleanвҖқеҹәжң¬зұ»еһӢзҡ„еҺҹеӯҗдҝ®ж”№ж“ҚдҪңзұ»AtomicBoolean
  • з¬¬дәҢзұ»пјҡAtomicStampedReferenceе’ҢAtomicMarkableReferenceгҖӮиҝҷдёӨдёӘзұ»иў«и®ҫи®Ўз”ЁжқҘи§ЈеҶіCASж“ҚдҪңзҡ„ABAй—®йўҳ

2.1.1гҖҒз¬¬дёҖзұ»

AtomicIntegerпјҢAtomicLongпјҢAtomicReferenceпјҢAtomicBooleanиҝҷ4дёӘзұ»зҡ„е®һзҺ°еӨ§еҗҢе°ҸејӮпјҢжҺҘдёӢжқҘд»ҘAtomicIntegerзұ»дёәдҫӢиҝӣиЎҢиҜҙжҳҺгҖӮ

AtomicIntegerжңүдёӨдёӘжҲҗе‘ҳеҸҳйҮҸпјҡ

  • е®һдҫӢжҲҗе‘ҳеҸҳйҮҸvalueпјҢж ёеҝғж“ҚдҪңеҸҳйҮҸ
  • зұ»жҲҗе‘ҳеҸҳйҮҸvalueOffsetпјҢе®ғдҝқеӯҳдәҶAtomicIntegerзұ»е®һдҫӢеҜ№иұЎзҡ„еҶ…еӯҳиө·е§Ӣең°еқҖеҲ°е…¶еҶ…valueеҸҳйҮҸеҶ…еӯҳиө·е§Ӣең°еқҖзҡ„еҒҸз§»еҖјпјҢд№ӢжүҖд»ҘйңҖиҰҒиҝҷдёӘеҒҸз§»еҖјжҳҜеӣ дёәжүҖж¶үеҸҠеҲ°зҡ„еҮ дёӘUnsafeзұ»дёӢж–№жі•жҳҜд»Ҙиҝҷж ·зҡ„ж–№ејҸеҺ»ж“ҚдҪңvalueеҸҳйҮҸзҡ„еҶ…еӯҳ

еңЁзұ»дёӯжҜ”иҫғйҮҚиҰҒзҡ„дёӨдёӘж–№жі•жҳҜaddAndGetе’ҢlazySetгҖӮ
1гҖҒaddAndGet(int delta)
AtomicIntegerзұ»дёӯзҡ„addAndGet(int delta)ж–№жі•гҖӮ

Unsafeзұ»дёӯзҡ„getAndAddInt(Object var1, long var2, int var4)ж–№жі•гҖӮ

з»“еҗҲgetIntVolatileж–№жі•е…·жңүvolatileеҸҳйҮҸеҶ…еӯҳиҜӯд№үе’ҢcompareAndSwapIntж–№жі•е…·жңүvoltileеҸҳйҮҸеҶ…еӯҳиҜӯд№үзҡ„еүҚеҜјзҹҘиҜҶпјҢеҸҜзҹҘжӯЈзЎ®е®һзҺ°вҖңеўһеҠ дё”иҝ”еӣһж“ҚдҪңвҖқзҡ„еҺҹеӯҗеҢ–гҖӮ


д»ҘдёҠжҳҜJDK 8зүҲжң¬зҡ„е®һзҺ°пјҢжҺҘдёӢжқҘеҜ№жҜ”JDK 6зүҲжң¬зҡ„е®һзҺ°гҖӮ
JDK 6дёӯпјҢAtomicIntegerзұ»дёӯзҡ„addAndGet(int delta)ж–№жі•гҖӮ

JDK 6дёӯпјҢAtomicIntegerзұ»дёӯзҡ„get()ж–№жі•гҖӮ

дёӨиҖ…жңҖеӨ§зҡ„дёҚеҗҢд№ӢеӨ„еңЁдәҺпјҡе…·жңүvolatileеҸҳйҮҸиҜ»иҜӯд№үйҷҗеҲ¶ең°иҜ»еҸ–valueеҸҳйҮҸпјҢиҷҪз„¶жңҖз»Ҳзӯүд»·пјҢдҪҶжҳҜдҪҝз”Ёзҡ„ж–№ејҸдёҚеҗҢвҖ”вҖ”JDK 8дҪҝз”ЁgetIntVolatileж–№жі•пјҢJDK 6зӣҙжҺҘиҝ”еӣһvalueеҸҳйҮҸпјҢиҖҢиҜҘеҸҳйҮҸжң¬е°ұз”ұvolatileдҝ®йҘ°гҖӮ

2гҖҒlazySet(int newValue)
еҹәдәҺUnsafeзұ»дёӢзҡ„putOrderedIntж–№жі•е®һзҺ°гҖӮж №жҚ®Unsafeзұ»дёӢputOrderedInt/putOrderedLong/putOrderedObjectж–№жі•зҡ„еҗ«д№үеүҚеҜјзҹҘиҜҶпјҢжҲ‘д»¬зҹҘйҒ“Unsafe.putOrderedInt(Object var1, long var2, int var4)д»ӢдәҺвҖңжҷ®йҖҡеҸҳйҮҸеҶҷиҜӯд№үвҖқе’ҢвҖңvolatileеҸҳйҮҸеҶҷиҜӯд№үвҖқд№Ӣй—ҙпјҢиҷҪз„¶дёҚе…·жңүvolatileеҸҳйҮҸеҶҷиҜӯд№үпјҢдҪҶжҳҜеңЁжңүдәӣеңәжҷҜдёӯиғҪеӨҹиҺ·еҫ—еҫҲеӨ§зҡ„жҖ§иғҪжҸҗеҚҮпјҢжҜ”еҰӮвҖңеӨ§йҮҸеҶҷд№ӢеҗҺпјҢеҶҚжңүдёҖдёӘиҜ»пјҢеҶҷе’ҢиҜ»д№Ӣй—ҙзҡ„еҸҜи§ҒжҖ§зЎ®дҝқз”ұе…¶д»–жңәеҲ¶жҸҗдҫӣпјҢдҫӢеҰӮвҖҳsynchronizedеҶ…еӯҳиҜӯд№үвҖҷпјҢвҖҳvolatileеҶ…еӯҳиҜӯд№үвҖҷзӯүвҖқгҖӮ

2.1.2гҖҒз¬¬дәҢзұ»

жҲ‘д»¬зҹҘйҒ“CASеҺҹеӯҗж“ҚдҪңеӯҳеңЁABAй—®йўҳпјҢи§ЈеҶіABAй—®йўҳзҡ„еёёи§Ғж–№жЎҲжҳҜеўһеҠ дёҖдёӘзүҲжң¬еҸ·пјҢеңЁjava.util.concurrent.atomicеҢ…дёӢжңүдёӨдёӘзұ»з”ЁдәҺи§ЈеҶіиҝҷдёӘй—®йўҳпјҡ

  • AtomicMarkableReferenceпјҢеј•е…ҘдёҖдёӘbooleanзұ»еһӢзҡ„ж Үи®°дҪҚпјҢдҪңдёәз®ҖеҢ–еҪўејҸзҡ„зүҲжң¬еҸ·пјҢд№ҹеҚіеҸӘжңүвҖңtrueвҖқе’ҢвҖңfalseвҖқдёӨдёӘзүҲжң¬еҸ·пјҢе®ғеҸӘиғҪи§ЈеҶіеҒ¶ж•°ABAй—®йўҳпјҢиҖҢдёҚиғҪи§ЈеҶіеҘҮж•°ABAй—®йўҳпјҢжҜ”еҰӮвҖңA-B-C-AпјҢеҒҮе®ҡж Үи®°дҪҚзҡ„еҸҳеҢ–дёәвҖҳtrue-false-true-falseвҖҷпјҢжӯӨж—¶(A,false)е№¶йқһйў„жңҹзҡ„(A,true)пјҢж•…дҝ®ж”№е°қиҜ•дјҡеӨұиҙҘпјӣA-B-AпјҢеҒҮе®ҡж Үи®°дҪҚзҡ„еҸҳеҢ–дёәвҖҳtrue-false-trueвҖҷпјҢжӯӨж—¶(A,true)дёәйў„жңҹзҡ„(A,true)пјҢж•…дҝ®ж”№е°қиҜ•дјҡжҲҗеҠҹвҖқ
  • AtomicStampedReferenceпјҢеј•е…ҘдёҖдёӘintзұ»еһӢзҡ„зүҲжң¬еҸ·пјҢжӯЈз»ҹеҪ»еә•зҡ„и§ЈеҶіж–№жЎҲпјҢж—ўиғҪи§ЈеҶіеҒ¶ж•°ABAй—®йўҳпјҢд№ҹиғҪи§ЈеҶіеҘҮж•°ABAй—®йўҳ

жҺҘдёӢжқҘе°ұдё»иҰҒд»Ӣз»ҚAtomicStampedReferenceзұ»гҖӮ

йҰ–е…Ҳд»Ӣз»Қе°ҒиЈ…еј•з”ЁеҸҳйҮҸreferenceе’ҢзүҲжң¬еҸ·еҸҳйҮҸstampзҡ„еҶ…йғЁзұ»PairпјҢе®ғд»¬йғҪз”ұfinalдҝ®йҘ°дё”йғҪеңЁжһ„йҖ ж–№жі•дёӯиөӢеҖјпјҢжҳҫиҖҢжҳ“и§Ғж»Ўи¶ідәҶfinalе…ій”®иҜҚеҶ…еӯҳеҸҜи§ҒиҜӯд№үи§„еҲҷзҡ„жқЎд»¶гҖӮ

з„¶еҗҺд»Ӣз»ҚcompareAndSet(V expectedReference, V newReference, int expectedStamp, int newStamp)иҝҷдёӘж ёеҝғж–№жі•пјҡ

дёҠиҝ°ж–№жі•зҡ„иҜӯд№үжҳҜпјҡеҸӘжңүеҪ“вҖңж—§еҖјвҖқе’ҢвҖңж—§зүҲжң¬еҸ·вҖқйғҪеҢ№й…Қж—¶пјҢжүҚиҝӣиЎҢвҖңж–°еҖјвҖқе’ҢвҖңж–°зүҲжң¬еҸ·вҖқзҡ„жӣҙж–°пјҢе…¶дёӯ(newReference == current.reference && newStamp == current.stamp)йҖ»иҫ‘иЎЁзӨәеңЁвҖңж—§еҖјвҖқе’ҢвҖңж–°еҖјвҖқпјҢвҖңж—§зүҲжң¬еҸ·вҖқе’ҢвҖңж–°зүҲжң¬еҸ·вҖқйғҪзӣёеҗҢзҡ„жғ…еҪўдёӢпјҢж— йңҖеҒҡе®һйҷ…зҡ„жӣҙж–°ж“ҚдҪңпјҢд»ҺиҖҢйҒҝе…Қж— еҝ…иҰҒзҡ„CASеҺҹеӯҗж“ҚдҪңгҖӮ

2.2гҖҒж•°з»„е…ғзҙ еҺҹеӯҗдҝ®ж”№ж“ҚдҪңзұ»

еҢ…жӢ¬3дёӘзұ»пјҡAtomicIntegerArrayпјҢAtomicLongArrayпјҢAtomicReferenceArrayгҖӮиҝҷ3дёӘзұ»зҡ„е®һзҺ°еӨ§еҗҢе°ҸејӮпјҢжҺҘдёӢжқҘд»ҘAtomicIntegerArrayзұ»дёәдҫӢиҝӣиЎҢиҜҙжҳҺгҖӮ

AtomicIntegerArrayжңү3дёӘжҲҗе‘ҳеҸҳйҮҸпјҡ

  • е®һдҫӢжҲҗе‘ҳеҸҳйҮҸint[] arrayпјҢж ёеҝғж“ҚдҪңеҸҳйҮҸгҖӮз”ұfinalдҝ®йҘ°пјҢеңЁжһ„йҖ ж–№жі•дёӯиөӢеҖјпјҢжҳҫиҖҢжҳ“и§Ғж»Ўи¶ідәҶfinalе…ій”®иҜҚеҶ…еӯҳеҸҜи§ҒиҜӯд№үи§„еҲҷзҡ„жқЎд»¶
  • зұ»жҲҗе‘ҳеҸҳйҮҸbaseпјҢиҝҷйҮҢдҝқеӯҳдәҶarrayж•°з»„еј•з”ЁеҜ№иұЎвҖңеҶ…еӯҳиө·е§Ӣең°еқҖвҖқдёҺвҖңж•°з»„еҜ№иұЎеҶ…е…ғзҙ еҶ…еӯҳиө·е§Ӣең°еқҖвҖқзҡ„еҒҸз§»йҮҸпјҢеҚіж•°з»„еј•з”ЁеҜ№иұЎеҜ№иұЎеӨҙзҡ„еӯ—иҠӮж•°гҖӮе®ғзҡ„и®Ўз®—ж–№жі•жҳҜUnsafe.arrayBaseOffset(int[].class)гҖӮйңҖиҰҒжіЁж„Ҹзҡ„жҳҜпјҢжҳҜеҗҰејҖеҗҜжҢҮй’ҲеҺӢзј©дјҡеҪұе“ҚиҜҘеҖј
  • зұ»жҲҗе‘ҳеҸҳйҮҸshiftпјҢе®ғзҡ„еҖјдёә2пјҢ2^2=4пјҢ4жҳҜintеһӢжүҖеҚ жҚ®зҡ„еӯ—иҠӮеӨ§е°ҸгҖӮе®ғзҡ„и®Ўз®—ж–№жі•жҳҜscale=Unsafe.arrayIndexScale(int[].class)е’Ңshift=31 - Integer.numberOfLeadingZeros(scale)пјҡеүҚиҖ…зЎ®е®ҡintеһӢжүҖеҚ жҚ®зҡ„еӯ—иҠӮеӨ§е°ҸпјҢеҗҺиҖ…и®Ўз®—д»Ҙ2дёәеә•зҡ„log(4)еҜ№ж•°еҖј

йҖҡиҝҮйҳ…иҜ»жәҗз ҒеҸҜд»ҘеҸ‘зҺ°пјҢеҜ№ж•°з»„дёӯжҹҗдёӘе…ғзҙ иҝӣиЎҢеҺҹеӯҗдҝ®ж”№ж“ҚдҪңзҡ„вҖңи·Ҝеҫ„вҖқжҳҜпјҡ

  1. е…Ҳе®ҡдҪҚиҜҘж•°з»„е…ғзҙ зҡ„еҶ…еӯҳең°еқҖ
  2. еҶҚи°ғз”ЁUnsafeзұ»дёӯзҡ„ж“ҚдҪңж–№жі•пјҢжҜ”еҰӮcompareAndSwapIntпјҢputIntVolatileзӯүгҖӮжң¬жӯҘе®һзҺ°дёҺвҖңеҹәжң¬зұ»еһӢжҲ–иҖ…еј•з”Ёзұ»еһӢеҺҹеӯҗдҝ®ж”№ж“ҚдҪңзұ»вҖқзҡ„е®һзҺ°е№¶ж— дәҢиҮҙпјӣеҗҢж—¶дҪҗиҜҒдәҶUnsafeзұ»дёӢCASж–№жі•пјҢput*Volatileе’Ңget*Volatileж–№жі•зҡ„volatileеҸҳйҮҸеҶ…еӯҳиҜӯд№үе№¶дёҚиҰҒжұӮеҸҳйҮҸжҳҜз”ұvolatileдҝ®йҘ°зҡ„пјҢеӣ дёәиҝҷйҮҢзҡ„ж•°з»„е…ғзҙ е№¶йқһз”ұvolatileдҝ®йҘ°зҡ„

еҸҰеӨ–жңүдёҖзӮ№еҖјеҫ—иҜҙжҳҺзҡ„жҳҜпјҡеңЁAtomicReferenceArrayзұ»дёӯжңүдёҖдёӘзӢ¬жңүзҡ„readObject()ж–№жі•пјҲдёҺвҖңAtomicIntegerArrayвҖқе’ҢвҖңAtomicLongArrayвҖқзұ»зӣёжҜ”пјүпјҢе®ғжҳҜдёәдәҶзЎ®дҝқеңЁеҸҚеәҸеҲ—еҢ–ж—¶дј е…Ҙзҡ„ж•°з»„зұ»еһӢзҡ„зЎ®жҳҜObject[]зұ»еһӢпјҢиҖҢйқһе…¶д»–иҷҪз„¶дёҚдјҡжҠҘй”ҷзҡ„ж•°з»„зұ»еһӢпјҢжҜ”еҰӮвҖңString[]вҖқпјҢиҝҷжҳҜдҪңдёәдёҖдёӘе®үе…ЁиЎҘдёҒиҖҢеј•е…Ҙзҡ„[1]пјӣиҖҢеңЁAtomicIntegerArray/AtomicLongArrayзұ»дёӯпјҢеҰӮжһңдј е…Ҙзҡ„ж•°з»„зұ»еһӢдёҚжҳҜint[]/long[]еҲҷдјҡз«ӢеҚіжҠҘй”ҷгҖӮ

2.3гҖҒеҜ№иұЎеӯ—ж®өеҺҹеӯҗдҝ®ж”№ж“ҚдҪңзұ»

еҢ…жӢ¬3дёӘзұ»пјҡAtomicIntegerFieldUpdaterпјҢAtomicLongFieldUpdaterпјҢAtomicReferenceFieldUpdaterгҖӮиҝҷ3дёӘзұ»зҡ„е®һзҺ°еӨ§еҗҢе°ҸејӮпјҢжҺҘдёӢжқҘд»ҘAtomicIntegerFieldUpdaterзұ»дёәдҫӢиҝӣиЎҢиҜҙжҳҺгҖӮ

йҖҡиҝҮйҳ…иҜ»жәҗз ҒеҸҜд»ҘеҸ‘зҺ°пјҢеҜ№еҜ№иұЎдёӯжҹҗдёӘеӯ—ж®өиҝӣиЎҢеҺҹеӯҗдҝ®ж”№ж“ҚдҪңзҡ„вҖңи·Ҝеҫ„вҖқжҳҜпјҡ

  1. е…Ҳе®ҡдҪҚеҜ№иұЎдёӯиҜҘеӯ—ж®өзҡ„еҶ…еӯҳең°еқҖ
  2. еҶҚи°ғз”ЁUnsafeзұ»дёӯзҡ„ж“ҚдҪңж–№жі•пјҢжҜ”еҰӮcompareAndSwapIntпјҢputIntVolatileзӯү

иҝҷйҮҢжңүдёӨзӮ№йңҖиҰҒиҜҙжҳҺпјҡ

  • еңЁиҝҷдёӘдҪҝз”ЁеңәжҷҜдёӯпјҢеӯ—ж®өзҡ„ж“ҚдҪңе…ҘеҸЈдёҚеҸӘйҷҗдәҺAtomicIntegerFieldUpdaterзұ»пјҢж•…еҪ“ж··жқӮе…¶д»–ж“ҚдҪңе…ҘеҸЈж—¶пјҢдёҚиғҪдҝқиҜҒжүҖж¶үеҸҠеӯ—ж®өж“ҚдҪңзҡ„жңүеәҸжҖ§гҖҒеҸҜи§ҒжҖ§е’ҢеҺҹеӯҗжҖ§гҖӮжҜ”еҰӮвҖңеҜ№дәҺдёҖдёӘеӯ—ж®өAпјҢдёҖеӨ„дҪҝз”ЁAtomicIntegerFieldUpdaterзұ»е®һдҫӢиҝӣиЎҢдҝ®ж”№пјҢеҸҰеӨ–дёҖеӨ„зӣҙжҺҘиҝӣиЎҢдҝ®ж”№вҖқ
  • еңЁAtomicIntegerFieldUpdaterзұ»зҡ„JavaDocдёӯиҰҒжұӮжүҖж“ҚдҪңеӯ—ж®өжҳҜз”ұvolatileдҝ®йҘ°зҡ„пјҢиҖҢдё”жһ„йҖ ж–№жі•д№ҹзӣҙжҺҘдҪңдәҶзӣёеә”зҡ„йҷҗе®ҡпјҢдҪҶе…¶е®һдҪҝз”ЁUnsafeзұ»дёӢCASж–№жі•пјҢput*Volatileе’Ңget*Volatileж–№жі•иҺ·еҫ—volatileеҸҳйҮҸеҶ…еӯҳиҜӯд№үе№¶дёҚиҰҒжұӮеҸҳйҮҸжҳҜз”ұvolatileдҝ®йҘ°зҡ„

2.4гҖҒеҺҹеӯҗи®Ўж•°/зҙҜеҠ ж“ҚдҪңзұ»

еҢ…жӢ¬5дёӘзұ»пјҡStriped64пјҢLongAdderпјҢLongAccumulatorпјҢDoubleAdderе’ҢDoubleAccumulatorгҖӮ
Striped64жҳҜзҲ¶зұ»пјҢLongAdder/DoubleAdderеҲҶеҲ«жҳҜ(LongAdderпјҢLongAccumulator)/(DoubleAdderпјҢDoubleAccumulator)зі»еҲ—зҡ„д»ЈиЎЁзұ»пјҢж•…жҺҘдёӢжқҘд»Ӣз»ҚвҖңStriped64вҖқпјҢвҖңLongAdderвҖқе’ҢвҖңDoubleAdderвҖқиҝҷ3дёӘзұ»гҖӮ

2.4.1гҖҒStriped64

еҹәжң¬еҺҹзҗҶжҳҜпјҡйҖҡиҝҮдёӨеӨ„иҝӣиЎҢи®Ўж•°/зҙҜеҠ пјҢдёҖеӨ„жҳҜдҪҝз”ЁbaseеҸҳйҮҸиҝӣиЎҢеҹәжң¬и®Ўж•°/зҙҜеҠ пјҢеҸҰеӨ–дёҖеӨ„жҳҜйҖҡиҝҮcellsж•°з»„иҝӣиЎҢеӨҡзәҝзЁӢеҲҶж®өи®Ўж•°/зҙҜеҠ гҖӮ

ж ёеҝғж–№жі•жҳҜlongAccumulateе’ҢdoubleAccumulateпјҢдёӨиҖ…зұ»дјјпјҢд»ҘlongAccumulateдёәдҫӢиҝӣиЎҢиҜҙжҳҺгҖӮ

дёҠиҝ°ж–№жі•зҡ„дё»дҪ“жөҒзЁӢжҳҜеҚҒеҲҶжё…жҷ°зҡ„пјҢе°ұжҳҜвҖңйҖҡиҝҮдёӨеӨ„иҝӣиЎҢи®Ўж•°/зҙҜеҠ вҖқпјҢдҪҶз”ұдәҺйңҖиҰҒиҖғиҷ‘еҗ„з§Қе№¶еҸ‘жғ…еҪўпјҢдё”дёәж— й”Ғзј–зЁӢе®һзҺ°ж–№жЎҲпјҲдҪҝз”ЁCASиҮӘж—Ӣй”ҒпјүпјҢеӣ жӯӨзӣёеҜ№жқҘиҜҙзҗҶи§Јиө·жқҘиҫғдёәиҙ№еҠІпјҢзүўи®°вҖңеӨҡзәҝзЁӢзҺҜеўғеӯҳеңЁеҗ„з§Қе№¶еҸ‘еҸҜиғҪпјҢе…ій”®ж“ҚдҪңйҖҡиҝҮCASж–№жі•иҝӣиЎҢвҖқиҝҷзӮ№еҸҜеё®еҠ©зҗҶи§ЈгҖӮ

жҺҘдёӢжқҘд»Ӣз»Қж–№жі•дёӯеҮ дёӘе…ій”®еҸҳйҮҸзҡ„еҗ«д№үпјҡ

  • cellsBusyеҸҳйҮҸз”ЁдҪңй”Ғз”ЁйҖ”пјҢcellsBusy=0ж—¶иЎЁзӨәжңӘй”ҒдҪҸжҖҒпјҢcellsBusy=1ж—¶иЎЁзӨәй”ҒдҪҸжҖҒпјҢдҪҝз”ЁCASж–№жі•ж“ҚдҪңиҜҘеҸҳйҮҸ
  • wasUncontendedеҸҳйҮҸпјҢз”ЁдәҺдёҖдёӘе°ҸжҖ§иғҪдјҳеҢ–пјҢеҪ“е…¶еҖјдёәfalseж—¶пјҢиЎЁзӨәеңЁи°ғз”ЁиҜҘж–№жі•еүҚе·Із»ҸеӨұиҙҘжү§иЎҢa.cas(v = a.value, ((fn == null) ? v + x : fn.applyAsLong(v, x)))зҡ„зӯүд»·иҜӯеҸҘпјҢж•…еҸҜдёҚжү§иЎҢеҗҺз»ӯжңҖдёҙиҝ‘дёҖж¬Ўзҡ„a.cas(v = a.value, ((fn == null) ? v + x : fn.applyAsLong(v, x)))иҜӯеҸҘ
  • еҪ“collideеҸҳйҮҸзҡ„еҖјдёәtrueзҡ„ж—¶еҖҷпјҢиЎЁзӨәе…Ғи®ёжү©еұ•cellsж•°з»„

2.4.2гҖҒLongAdder

Longзұ»еһӢзҡ„и®Ўж•°зұ»пјҢж ёеҝғж–№жі•еңЁзҲ¶зұ»Striped64дёӯпјҢжң¬иә«дё»иҰҒжҸҗдҫӣиҺ·еҸ–и®Ўж•°з»“жһң/йҮҚзҪ®и®Ўж•°зҡ„ж–№жі•пјҢйңҖиҰҒжіЁж„Ҹзҡ„жҳҜпјҢе®ғдёҚжҳҜзәҝзЁӢе®үе…Ёзҡ„и®Ўж•°ж•°жҚ®з»“жһ„пјҢеҺҹеӣ еҸӮи§ҒеҰӮдёӢжәҗз ҒгҖӮ

2.4.3гҖҒDoubleAdder

Doubleзұ»еһӢзҡ„и®Ўж•°зұ»пјҢи·ҹLongAddrзұ»дјјгҖӮдёҖдёӘе®һзҺ°иҰҒзӮ№жҳҜпјҡе°ҶDoubleзұ»еһӢеӯҳеӮЁжҲҗLongзұ»еһӢпјҢеҸҜж“ҚдҪңзҡ„еҹәзЎҖжҳҜдёӨиҖ…йғҪжҳҜ8дёӘеӯ—иҠӮеӨ§е°ҸгҖӮ
