六死八活七半

发表于 2023-12-15 更新于 2024-04-15 专题于 围棋 Disqus：

棋形，是棋子在棋盘上组合成的各种形状。
本文介绍3种基础死活棋形——“六死八活七半”。


在介绍之前先作两点说明：

  1. 双方都按照最强落子原则进行探讨，否则毫无实际意义
  2. 本文案例讨论中以黑棋为守方，白棋为攻方。故：
    • 黑子的目标是：尽量能做活
    • 白子的目标是：尽量能阻止黑棋做活

一、死活棋形

1.1、死形之“六死”

守方必不能做活，故“攻守双方都无需落子”。

具体分析过程（轴对称，基于A、B和C点进行探讨，其他属于对称情形）分为“黑先”和“白先”两种情形。

1、黑先
黑必不能做活。

  1. 黑A -> 白F：黑做活失败
  2. 黑B -> 白F：黑做活失败
  3. 黑C -> 白F：黑做活失败
  4. [与1对称]黑F -> 白A：黑做活失败
  5. [与2对称]黑E -> 白A：黑做活失败
  6. [与3对称]黑D -> 白A：黑做活失败

2、白先
黑必不能做活：无需先落子，黑落子后必能按照“1、黑先”小节阻止黑做活。

1.2、活形之“八活”

守方必能做活，故“攻守双方都无需落子”。

具体分析过程（轴对称，基于A、B、C和D点进行探讨，其他属于对称情形）分为“黑先”和“白先”两种情形。

1、黑先
黑必能做活（存在一种做活下法即可）：

  1. 黑A -> 白H：黑做活成功
  2. 黑B -> 白H：黑做活成功
  3. 黑C -> 白H：黑做活成功
  4. 黑D -> 白H：黑做活成功
  5. [与1对称]黑H -> 白A：黑做活成功
  6. [与2对称]黑G -> 白A：黑做活成功
  7. [与3对称]黑F -> 白A：黑做活成功
  8. [与4对称]黑E -> 白A：黑做活成功

2、白先
黑必能做活（没有一种下法能够成功阻止做活）：

  1. 白A -> 黑B：阻止黑做活失败
  2. 白B -> 黑C：阻止黑做活失败
  3. 白C -> 黑D：阻止黑做活失败
  4. 白D -> 黑E：阻止黑做活失败
  5. [与1对称]白H -> 黑G：阻止黑做活失败
  6. [与2对称]白G -> 黑F：阻止黑做活失败
  7. [与3对称]白F -> 黑E：阻止黑做活失败
  8. [与4对称]白E -> 黑D：阻止黑做活失败

1.3、半死不活形之“七半”

守方先落子，存在一种下法能成功做活；攻方先落子，存在一种下法能够成功阻止做活。

结论：攻方先，抢到A和G点，成功阻止做活；守方先，抢到A、D和G点，成功做活。为了减少记忆成本，把关键点统一记为A和G。

具体分析过程（轴对称，基于A、B、C和D点进行探讨，其他属于对称情形）分为“黑先”和“白先”两种情形。

1、黑先
黑必能做活（存在一种做活下法即可）：

  1. 黑A -> 白G -> 黑F：黑做活成功
  2. 黑B -> 白G：黑做活失败
  3. 黑C -> 白A：黑做活失败
  4. 黑D -> 黑后续只要一边做一只真眼：黑做活成功
  5. [与1对称]黑G -> 白A -> 黑B：黑做活成功
  6. [与2对称]黑F -> 白A：黑做活失败
  7. [与3对称]黑E -> 白G：黑做活失败

2、白先
黑必不能做活（存在一种能够成功阻止做活下法即可）：

  1. 白A -> 黑G：阻止黑做活成功
  2. 白B -> 黑A：阻止黑做活失败
  3. 白C -> 黑D：阻止黑做活失败
  4. 白D：阻止黑做活失败
  5. [与1对称]白G -> 黑A：阻止黑做活成功
  6. [与2对称]白F -> 黑G：阻止黑做活失败
  7. [与3对称]白E -> 黑D：阻止黑做活失败

二、死活棋形之劫材

劫材再回顾下是“彼之弱点，我之劫材”。

2.1、死形

死形的劫材如果存在必属于守方：守方落子 -> 攻方必须应劫，否则守方做活成功。

结论：A、C、D和F四个点都是守方的劫材。

具体分析过程：

  1. 黑A -> 白必须应劫落子F，否则黑做活成功
  2. 黑B -> 白无须立即落子应对：不是黑的劫材
  3. 黑C -> 白必须应劫落子A，否则黑做活成功
  4. [与1对称]黑F -> 白必须应劫落子A，否则黑做活成功
  5. [与2对称]黑E -> 白无须立即落子应对：不是黑的劫材
  6. [与3对称]黑D -> 白必须应劫落子F，否则黑做活成功

2.2、活形

活形的劫材如果存在必属于攻方：攻方落子 -> 守方必须应劫，否则攻方阻止做活成功。

结论：A和H两个点是攻方的劫材。

具体分析：

  1. 白A -> 黑必须立即落子应对，否则白继续落子H，黑做活失败：是白的劫材
  2. 白B -> 黑无须立即落子应对：不是白的劫材
  3. 白C -> 黑无须立即落子应对：不是白的劫材
  4. 白D -> 黑无须立即落子应对：不是白的劫材
  5. [与1对称]白H -> 黑必须立即落子应对，否则白继续落子A，黑做活失败：是白的劫材
  6. [与2对称]白G -> 黑无须立即落子应对：不是白的劫材
  7. [与3对称]白F -> 黑无须立即落子应对：不是白的劫材
  8. [与4对称]白E -> 黑无须立即落子应对：不是白的劫材

2.3、半死不活形

按照劫材定义——“逼迫对方应对”，半死不活形没有劫材，因为：

  1. 守方先
    • 如果抢占做活关键点成功做活，无法逼迫对方应对
    • 如果不下做活关键点只能引诱（而不是逼迫）攻方抢占关键点阻止做活
  2. 攻方先
    • 如果抢占做活关键点成功阻止做活，无法逼迫对方应对
    • 如果不下做活关键点引诱（而不是逼迫）守方抢占关键点成功做活

但是也可将劫材定义广义化，将以上引诱作为劫材情形。

三、总结

棋形 死活类型 守方 攻方 劫材
六死 死形 无需主动落子 无需主动落子 A、C、D和F属于守方的劫材点
八活 活形 无需主动落子 无需主动落子 A和H两个点都是攻方的劫材点
七半 半死不活形 立即做活，关键点是边界两点 立即阻止做活，关键点是边界两点 边界两点是广义的劫材点
