灯具选择

发表于 2023-02-01 更新于 2024-10-23 专题于 生活 Disqus：

一、光源

光源是灯具的核心，故首先介绍光源。根据发光原理，可分为3类：1）白炽现象发光；2）荧光发光；3）LED发光。

关于光的一些介绍，可参见《光》。

1.1、白炽现象发光

发光原理：通电加热灯丝到白炽状态，利用热辐射发出可见光。

常见的基于白炽发光的灯具有：

  • 白炽灯，就是小时候用的那种时间长了会很烫手、经常呼啦一下烧断了钨丝的灯泡
  • 卤素灯，可以看做是白炽灯的升级版，会在灯泡内注入碘或溴等卤素气体，高温下，升华的钨丝与卤素进行化学作用，冷却后的钨会重新凝固在钨丝上，形成平衡的循环，避免钨丝过早断裂

该类灯具特点（基于该类常见灯具的定性描述，即可能存在该类不常见灯具违背以下特点）：

  1. 能源利用率：能源利用率低，所消耗的电能只有不到10%可转化为光能，其余部分以热能的形式散失
  2. 材料寿命：寿命短，1000小时左右
  3. 光危害：蓝光含量少，紫外线含量少，红外线含量高
  4. 色温：2500-3000K
  5. 显色指数：显色指数非常高，在95-100之间
  6. 照度：跟“灯具瓦数”和“能源利用率”相关

[备注] 我国从2011年起逐步实行淘汰计划，到2016年10月1日起，禁止销售和进口15瓦及以上普通照明用白炽灯。

1.2、荧光发光

发光原理：利用低气压的汞蒸气在通电后释放紫外线，从而使荧光粉发出可见光。

常见的基于荧光发光的灯具有：

  • 普通荧光灯，比如“小时候常见的长条状的日光灯就属于普通荧光灯的一种”
  • 节能灯（国外简称为CFL灯），全称为“紧凑型三基色电子荧光灯”，是一种紧凑型、自带镇流器的日光灯

该类灯具特点（基于该类常见灯具的定性描述，即可能存在该类不常见灯具违背以下特点）：

  1. 能源利用率：能源利用率低，所消耗的电能只有不到30%可转化为光能，其余部分以热能的形式散失
  2. 材料寿命：寿命短，6000小时左右
  3. 光危害：蓝光含量多，紫外线含量多，红外线含量少
  4. 色温：2700-6400K
  5. 显色指数：显色性差异较大，多数不高
  6. 照度：跟“灯具瓦数”和“能源利用率”相关

1.3、LED发光

发光二极管，是一种能够将电能转化为可见光的固态的半导体器件，它可以直接把电转化为光。英文名为Light Emitting Diode，简称LED。

该类灯具特点（基于该类常见灯具的定性描述，即可能存在该类不常见灯具违背以下特点）：

  1. 能源利用率：能源利用率高，所消耗的电能有80%以上可转化为光能，其余部分以热能的形式散失。故常被称为LED节能灯，须注意与“节能灯（CFL灯）”的区分
  2. 材料寿命：寿命长，5万-10万小时
  3. 光危害：蓝光含量多，紫外线含量少，红外线含量少
  4. 色温：范围广
  5. 显色指数：范围广
  6. 照度：跟“灯具瓦数”和“能源利用率”相关

[备注] 各国大力推广发展LED灯。

二、灯具选择

灯具选择的核心指标有：能源利用率，材料寿命，显色指数，色温，光危害，照度，无可视频闪；非核心指标有：智能控制，智能互动，更高安全性等。

首先须选择光源，结合上一章节内容，选择LED灯具，因为：LED灯具在可控制调节获得“高显色指数，合适色温，最小光危害，高照度，无可视频闪”的前提下，具有最高的能源利用率和最优的材料寿命表现。

接下来描述不同场景下的LED灯具选择。

2.1、工作学习场景

对除了“能源利用率”和“材料寿命”之外的核心指标要求如下：

  • 色温：4000K左右
  • 显色指数：大于80
  • 光危害：LED天然没有红外线、紫外线危害，控制减少蓝光危害
  • 照度：国AA级[3]
  • 无可视频闪

2.2、卧室（不工作学习）场景

对除了“能源利用率”和“材料寿命”之外的核心指标要求如下：

  • 色温：2700-3000K
  • 显色指数：不必要大于80
  • 光危害：LED天然没有红外线、紫外线危害，控制减少蓝光危害
  • 照度：没有太高要求
  • 无可视频闪
