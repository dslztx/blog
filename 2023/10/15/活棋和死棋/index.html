1.2、双真眼活棋

最常见的活棋棋形为：双真眼相连接（包括直接连接 + 间接连接）构成的棋形。

眼可分为3种：

  • 角上眼：1只眼角
  • 边上眼：2只眼角
  • 中央眼：4只眼角

在继续介绍之前，首先复习关于充分条件、必要条件和逆否命题的相关知识：

  1. 充分条件和必要条件：
    • 由A可以推出B（A -> B），由B可以推出A（B -> A），则A是B的充分必要条件（A <-> B）
    • 由A可以推出B（A -> B），由B不可以推出A，则A是B的充分不必要条件
    • 由A不可以推出B，由B可以推出A（B -> A），则A是B的必要不充分条件
    • 由A不可以推出B，由B不可以推出A，则A是B的既不充分也不必要条件
  2. 命题A -> B成立，则其逆否命题!B -> !A也成立

1.2.1、真眼与假眼定义

准确本质定义是：

  1. 假眼 <-> 能局部打吃的眼
  2. 互为逆否的结论：不能局部打吃（不能打吃或者只能整体打吃）的眼 <-> 真眼

1.2.2、判断方法

1.2.2.1、眼边连接判断法

结论：

  1. 眼边连接（直接连接 + 间接连接） <-> 真眼
  2. 互为逆否的结论：假眼 <-> 眼边不连接（直接连接 + 间接连接）

第1点的证明：

  • 充分性证明，即眼边连接（直接连接 + 间接连接） -> 真眼：各种连接棋形必能进展到不能打吃或者只能整体打吃，得证
  • 必要性证明，即真眼 -> 眼边连接（直接连接 + 间接连接）：真眼（不能打吃或者只能整体打吃）必能推导到各种连接情形，得证
1.2.2.2、眼角判断法

眼边连接判断法计算量过大，故产生一种脱胎于眼边连接判断法（敌方占据眼角是某种程度的分断）的减小计算量的近似判断法——眼角判断法。

其结论为：

  1. 角上眼（共1只眼角）：
    • 被敌方占据1只眼角是假眼[近似]
    • 未被敌方占据1只眼角（1只眼角或为己方所占，或不为双方所占）是真眼[准确]
  2. 边上眼（共2只眼角）：
    • 被敌方占据>=1只眼角是假眼[近似]
    • 未被敌方占据>=1只眼角（2只眼角或为己方所占，或不为双方所占）是真眼[准确]
  3. 中央眼（共4只眼角）：
    • 被敌方占据>=2只眼角是假眼[近似]
    • 未被敌方占据>=2只眼角（至少3只眼角或为己方所占，或不为双方所占）是真眼[准确]

以上判断法解析：

  1. 角上眼被敌方占据1只眼角 / 边上眼被敌方占据>=1只眼角 / 中央眼被敌方占据>=2只眼角是相应眼不连接（直接连接 + 间接连接）的必要不充分条件（如图3、图4和图5中眼A），而不连接（直接连接 + 间接连接） <-> 假眼，故有角上眼被敌方占据1只眼角 / 边上眼被敌方占据>=1只眼角 / 中央眼被敌方占据>=2只眼角是相应眼是假眼的必要不充分条件，即：
    • 角上眼被敌方占据1只眼角 / 边上眼被敌方占据>=1只眼角 / 中央眼被敌方占据>=2只眼角 -\-> 相应眼是假眼，因此以上判断法中的假眼判断是近似而不准确的
    • 相应眼是假眼 -> 角上眼被敌方占据1只眼角 / 边上眼被敌方占据>=1只眼角 / 中央眼被敌方占据>=2只眼角
  2. 第1点结论的逆否命题分别为：
    • 相应眼是真眼 -\-> 角上眼未被敌方占据1只眼角 / 边上眼未被敌方占据>=1只眼角 / 中央眼未被敌方占据>=2只眼角
    • 角上眼未被敌方占据1只眼角 / 边上眼未被敌方占据>=1只眼角 / 中央眼未被敌方占据>=2只眼角 -> 相应眼是真眼，因此以上判断法中的真眼判断是准确的
