12种基础死活棋形

发表于 2023-11-19 更新于 2024-03-14 专题于 围棋 Disqus：

棋形，是棋子在棋盘上组合成的各种形状。
本文介绍12种基础死活棋形，包括：两种死形，四种活形，六种半死不活形。


在介绍之前先作两点说明：

  1. 双方都按照最强落子原则进行探讨，否则毫无实际意义
  2. 本文案例讨论中以黑棋为守方，白棋为攻方。故：
    • 黑子的目标是：尽量能做活
    • 白子的目标是：尽量能阻止黑棋做活

一、死活棋形

1.1、死形

守方必不能做活，故“攻守双方都无需落子”。

1.1.1、直二

具体分析过程分为“黑先”和“白先”两种情形。

1、黑先
黑必不能做活。

2、白先
黑必不能做活。

1.1.2、方四

具体分析过程（中心对称，基于A点进行探讨，其他属于对称情形）分为“黑先”和“白先”两种情形。

1、黑先
黑必不能做活：

  1. 黑A -> 白D：黑做活失败
  2. [与1对称]黑B -> 白C：黑做活失败
  3. [与1对称]黑C -> 白B：黑做活失败
  4. [与1对称]黑D -> 白A：黑做活失败

2、白先
黑必不能做活：无需先落子，黑落子后必能按照“1、黑先”小节阻止黑做活。

1.2、活形

守方必能做活，故“攻守双方都无需落子”。

1.2.1、直四

具体分析过程（轴对称，基于A和B点进行探讨，其他属于对称情形）分为“黑先”和“白先”两种情形。

1、黑先
黑必能做活（存在一种做活下法即可）：

  1. 黑A -> 直三且白先：黑做活失败
  2. [与1对称]黑D -> 直三且白先：黑做活失败
  3. 黑B：黑做活成功
  4. [与3对称]黑C：黑做活成功

2、白先
黑必能做活（没有一种下法能够成功阻止做活）：

  1. 白A -> 黑B/C：阻止黑做活失败
  2. [与1对称]白D -> 黑B/C：阻止黑做活失败
  3. 白B -> 黑C：阻止黑做活失败
  4. [与3对称]白C -> 黑B：阻止黑做活失败

1.2.2、曲四

具体分析过程分为“黑先”和“白先”两种情形。

1、黑先
黑必能做活（存在一种做活下法即可）：

  1. 黑A -> 直三且白先：黑做活失败
  2. 黑B -> 白C/D：黑做活成功
  3. 黑C -> 白A/B：黑做活成功
  4. 黑D -> 弯三且白先：黑做活失败

2、白先
黑必能做活（没有一种下法能够成功阻止做活）：

  1. 白A -> 黑B：阻止黑做活失败
  2. 白B -> 黑C：阻止黑做活失败
  3. 白C -> 黑B：阻止黑做活失败
  4. 白D -> 黑C：阻止黑做活失败

1.2.3、闪电四

具体分析过程（中心对称，基于A和B点进行探讨，其他属于对称情形）分为“黑先”和“白先”两种情形。

1、黑先
黑必能做活（存在一种做活下法即可）：

  1. 黑A -> 弯三棋形且白先：黑做活失败
  2. [与1对称]黑D -> 弯三棋形且白先：黑做活失败
  3. 黑B：黑做活成功
  4. [与3对称]黑C：黑做活成功

2、白先
黑必能做活（没有一种下法能够成功阻止做活）：

  1. 白A -> 黑B/C：阻止黑做活失败
  2. [与1对称]白D -> 黑C/B：阻止黑做活失败
  3. 白B -> 黑C：阻止黑做活失败
  4. [与3对称]白C -> 黑B：阻止黑做活失败

1.2.4、板六

具体分析过程（中心对称，基于A和B点进行探讨，其他属于对称情形）分为“黑先”和“白先”两种情形。

1、黑先
黑必能做活（存在一种做活下法即可）：

  1. 黑A -> 刀把五且白先：黑做活失败
  2. [与1对称]黑C -> 刀把五且白先：黑做活失败
  3. [与1对称]黑D -> 刀把五且白先：黑做活失败
  4. [与1对称]黑F -> 刀把五且白先：黑做活失败
  5. 黑B：黑做活成功
  6. [与5对称]黑E：黑做活成功

2、白先
黑必能做活（没有一种下法能够成功阻止做活）：

  1. 白A -> 黑B/E：阻止黑做活失败
  2. [与1对称]白C -> 黑B/E：阻止黑做活失败
  3. [与1对称]白D -> 黑B/E：阻止黑做活失败
  4. [与1对称]白F -> 黑B/E：阻止黑做活失败
  5. 白B -> 黑E：阻止黑做活失败
  6. [与5对称]白E -> 黑B：阻止黑做活失败

1.3、半死不活形

守方先落子，存在一种下法能成功做活；攻方先落子，存在一种下法能够成功阻止做活。

1.3.1、直三

结论：抢B点——守方抢到B，成功做活；攻方抢到B，成功阻止做活。

具体分析过程（轴对称，基于A和B点进行探讨，其他属于对称情形）分为“黑先”和“白先”两种情形。

1、黑先
黑必能做活（存在一种做活下法即可）：

  1. 黑A：黑做活失败
  2. [与1对称]黑C：黑做活失败
  3. 黑B：黑做活成功

2、白先
黑必不能做活（存在一种能够成功阻止做活下法即可）：

  1. 白A -> 黑B：阻止黑做活失败
  2. [与1对称]白C -> 黑B：阻止黑做活失败
  3. 白B：阻止黑做活成功

1.3.2、弯三

结论：抢B点——守方抢到B，成功做活；攻方抢到B，成功阻止做活。

具体分析过程（轴对称，基于A和B点进行探讨，其他属于对称情形）分为“黑先”和“白先”两种情形。

1、黑先
黑必能做活（存在一种做活下法即可）：

  1. 黑A：黑做活失败
  2. [与1对称]黑C：黑做活失败
  3. 黑B：黑做活成功

2、白先
黑必不能做活（存在一种能够成功阻止做活下法即可）：

  1. 白A -> 黑B：阻止黑做活失败
  2. [与1对称]白C -> 黑B：阻止黑做活失败
  3. 白B：阻止黑做活成功

1.3.3、丁四

结论：抢C点——守方抢到C，成功做活；攻方抢到C，成功阻止做活。

具体分析过程（轴对称，基于A、B和C点进行探讨，其他属于对称情形）分为“黑先”和“白先”两种情形。

1、黑先
黑必能做活（存在一种做活下法即可）：

  1. 黑A -> 直三且白先：黑做活失败
  2. 黑B -> 弯三且白先：黑做活失败
  3. [与2对称]黑D -> 弯三且白先：黑做活失败
  4. 黑C：黑做活成功

2、白先
黑必不能做活（存在一种能够成功阻止做活下法即可）：

  1. 白A -> 黑C：阻止黑做活失败
  2. 白B -> 黑C：阻止黑做活失败
  3. [与2对称]白D -> 黑C：阻止黑做活失败
  4. 白C：阻止黑做活成功

1.3.4、刀把五

结论：攻方先，抢到B点（刀把），成功阻止做活；守方先，抢到A、B和E点，成功做活。为了减少记忆成本，把关键点统一记为B。

具体分析过程分为“黑先”和“白先”两种情形。

1、黑先
黑必能做活（存在一种做活下法即可）：

  1. 黑A -> 闪电四活形：黑做活成功
  2. 黑B：黑做活成功
  3. 黑C -> 方四：黑做活失败
  4. 黑D -> 丁四且白先：黑做活失败
  5. 黑E -> 曲四活形：黑做活成功

2、白先
黑必不能做活（存在一种能够成功阻止做活下法即可）：

  1. 白A/D/E/C -> 黑B：阻止黑做活失败
  2. 白B：阻止黑做活成功

1.3.5、梅花五

结论：攻方先，抢到C点，成功阻止做活；守方先，抢到C点，成功做活。

具体分析过程（轴对称，基于A和C点进行探讨，其他属于对称情形）分为“黑先”和“白先”两种情形。

1、黑先
黑必能做活（存在一种做活下法即可）：

  1. 黑A -> 白C：黑做活失败
  2. [与1对称]黑B -> 白C：黑做活失败
  3. [与1对称]黑D -> 白C：黑做活失败
  4. [与1对称]黑E -> 白C：黑做活失败
  5. 黑C：黑做活成功

2、白先
黑必不能做活（存在一种能够成功阻止做活下法即可）：

  1. 白A -> 黑C：阻止黑做活失败
  2. [与1对称]白B -> 黑C：阻止黑做活失败
  3. [与1对称]白D -> 黑C：阻止黑做活失败
  4. [与1对称]白E -> 黑C：阻止黑做活失败
  5. 白C：阻止黑做活成功

1.3.6、葡萄六

结论：攻方先，抢到D点，成功阻止做活；守方先，抢到A、D和E点，成功做活。为了减少记忆成本，把关键点统一记为D。

具体分析过程（轴对称，基于A、B、C和D点进行探讨，其他属于对称情形）分为“黑先”和“白先”两种情形。

1、黑先
黑必能做活（存在一种做活下法即可）：

  1. 黑A：黑做活成功
  2. [与1对称]黑E：黑做活成功
  3. 黑B -> 梅花五且白先：黑做活失败
  4. 黑C -> 刀把五且白先：黑做活失败
  5. [与4对称]黑F -> 刀把五且白先：黑做活失败
  6. 黑D：黑做活成功

2、白先
黑必不能做活（存在一种能够成功阻止做活下法即可）：

  1. 白A/B/C/E/F -> 黑D：阻止黑做活失败
  2. 白D：阻止黑做活成功

二、死活棋形之劫材

劫材再回顾下是“彼之弱点，我之劫材”。

2.1、死形

死形的劫材如果存在必属于守方：守方落子 -> 攻方必须应劫，否则守方做活成功。

2.1.1、直二

结论：守方无劫材。

2.1.2、方四

结论：A、B、C和D四个点都是守方的劫材。

具体分析过程：

  1. 黑A -> 白必须应劫落子D，否则黑做活成功
  2. 黑B -> 白必须应劫落子C，否则黑做活成功
  3. 黑C -> 白必须应劫落子B，否则黑做活成功
  4. 黑D -> 白必须应劫落子A，否则黑做活成功

2.2、活形

活形的劫材如果存在必属于攻方：攻方落子 -> 守方必须应劫，否则攻方阻止做活成功。

2.2.1、直四

结论：B和C两个点是攻方的劫材。

具体分析：

  1. 白A -> 黑无须立即落子应对：不是白的劫材
  2. [与1对称]白D -> 黑无须立即落子应对：不是白的劫材
  3. 白B -> 黑须立即落子C应对：是白的劫材
  4. [与3对称]白C -> 黑须立即落子B应对：是白的劫材

2.2.2、曲四

结论：B和C两个点是攻方的劫材。

具体分析：

  1. 白A -> 黑无须立即落子应对：不是白的劫材
  2. 白B -> 黑须立即落子C应对：是白的劫材
  3. [与2对称]白C -> 黑须立即落子B应对：是白的劫材
  4. [与1对称]白D -> 黑无须立即落子应对：不是白的劫材

2.2.3、闪电四

结论：B和C两个点是攻方的劫材。

具体分析：

  1. 白A -> 黑无须立即落子应对：不是白的劫材
  2. 白B -> 黑须立即落子C应对：是白的劫材
  3. 白C -> 黑须立即落子B应对：是白的劫材
  4. 白D -> 黑无须立即落子应对：不是白的劫材

2.2.4、板六

结论：B和E两个点是攻方的劫材。

具体分析：

  1. 白A -> 黑无须立即落子应对：不是白的劫材
  2. [与1对称]白C -> 黑无须立即落子应对：不是白的劫材
  3. [与1对称]白D -> 黑无须立即落子应对：不是白的劫材
  4. [与1对称]白F -> 黑无须立即落子应对：不是白的劫材
  5. 白B -> 黑须立即落子C应对：是白的劫材
  6. [与5对称]白E -> 黑须立即落子B应对：是白的劫材

2.3、半死不活形

按照劫材定义——“逼迫对方应对”，半死不活形没有劫材，因为：

  1. 守方先
    • 如果抢占做活关键点成功做活，无法逼迫对方应对
    • 如果不下做活关键点只能引诱（而不是逼迫）攻方抢占关键点阻止做活
  2. 攻方先
    • 如果抢占做活关键点成功阻止做活，无法逼迫对方应对
    • 如果不下做活关键点引诱（而不是逼迫）守方抢占关键点成功做活

但是也可将劫材定义广义化，将以上引诱作为劫材情形。
