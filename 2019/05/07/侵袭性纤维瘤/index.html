侵袭性纤维瘤

发表于 2019-05-07 更新于 2024-03-17 专题于 生活 Disqus：

一、肿瘤分类

肿瘤的分类和命名非常复杂，下面是一个简化的分类方法：

备注：

  • 交界性肿瘤：组织形态和生物学行为介于“良性肿瘤”和“恶性肿瘤”之间

1.1、恶性肿瘤特点

  • 生长失去控制
  • 容易复发
  • 局部浸润。恶性肿瘤的最基本特征之一，指侵蚀和破坏临近的与当前组织器官不同的组织器官。比如“如果乳腺癌细胞只在乳房中生长，那就没有发生局部浸润，如果侵蚀乳腺的皮肤，肋骨组织，胸壁组织等，那就发生了局部浸润”，“如果直肠癌细胞只在直肠中生长，那就没有发生局部浸润，如果侵蚀前列腺，膀胱组织等，那就发生了局部浸润”
  • 远处转移。恶性肿瘤的最基本特征之一，指远处转移到全身其他组织器官，与“局部浸润”相比，“局部浸润”强调“局部”，“远处转移”强调“远处”，远处转移有3种途径：1）血行转移；2）淋巴结转移；3）种植转移。“是否远处转移”是区分“恶性肿瘤”和“良性肿瘤”最为核心的特征
  • 主要危害在于会破坏组织器官。比如会造成“出血”，“坏死”，“溃疡”等

1.2、良性肿瘤特点

  • 生长速度缓慢
  • 不易复发
  • 不会局部浸润
  • 不会远处转移
  • 主要危害在于会压迫和阻塞组织器官，但不会破坏组织器官。比如“长在肠子里的良性肿瘤可导致肠梗阻”，“长在大脑里的良性肿瘤会压迫神经导致严重的后果”
  • 少数的良性肿瘤在一定条件下，可能会恶变为恶性肿瘤，因此，对于良性肿瘤还是不能掉以轻心

二、侵袭性纤维瘤

侵袭性纤维瘤（aggressive fibromatosis，AF），或称“硬纤维瘤（desmoid tumor，DT）”，或称“韧带样型纤维瘤（desmoid type fibromatosis，DTFT）”，或称“肌腱膜纤维瘤（musculoaponeurotic fibromatosis，MF）”，是成纤维细胞克隆性增生性病变，位于深部软组织，较少见，极难根治，具有浸润性生长、局部易复发（即意味着生长快）、无远处转移、罕见恶变（国内仅有1例恶变为纤维肉瘤的报道，该例恶变考虑可能与反复手术刺激、放射治疗有关）的特点，根据上述特点可知属于“交界性肿瘤”，轻度恶性。

据统计，每年在百万人口中可发生3-5个新病例，侵袭性纤维瘤5年内复发率高达50%左右，而且在术后2年内最容易出现复发，复发最迟可到术后12年，因此，有学者建议病人术后前3年内应该每3月复查，之后两年每半年复查，再之后一年复查一次。

三、几个问题

  • 不会远处转移？
  • 对于恶变概率，是小概率事件吗？
  • 因为不会远处转移下次检查只需在原部位做个B超？
  • 阴性切缘？
  • 各类活检（包括穿刺活检）均有促使恶性肿瘤扩散的潜在可能性，需要在术前短期内或术中施行活检？

四、侵袭性纤维瘤的最新进展

  • 全国侵袭性纤维瘤诊疗研究中心在河北燕达医院成立[4]
  • 一种新的手术治疗方式——海扶聚焦超声手术治疗，该治疗方式的最大特色是“无创”
